What is the main difference between CSS Grid and CSS Flex?

CSS Grid follows an authoritarian parenting where the parent class or tag tells the child tag what size to be. It follows a 2-dimensional grid layout. CSS Flex follows a 1-dimensional layout but works well for 2-D as well.
