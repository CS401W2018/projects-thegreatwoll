What is normal flow in HTML?

Normal flow in HTML refers to the normal order of tags or blocks and how they are arranged within the webpage.

What does float do? Why do you need to clear the float?

Float allows us to move an element within the page with other elements wrapping around it. Clearing the float breaks the floating tag or element from affecting the elements beneath it.

What do absolute, fixed, and relative do?

Relative allows us to slide an element around the page but reserves the original spot of the element. Absolute does not reserve the space in the webpage and it makes the element get on top of other elements. The good use of absolute are for the headers as it scrolls. Fixed allows the element to stay on the same spot even when scrolling.

How can you make nice navigation buttons for mobile devices?

We can make nice navigation buttons for mobile devices by allowing them to be in fixed positions so that even when the user scrolls through the page, the navigation buttons will still be visible and in the same position as it originally was at.
