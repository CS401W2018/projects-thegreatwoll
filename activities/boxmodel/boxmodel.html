What is the difference between block tags and inline tags?

Block Tags form a box and creates a line break and occupies 100% of the width. The height only depends on the content of the tag. Inline Tags are tags that does not cause line breaks. They are used within the lines to group certain elements and allow for selecting specific parts to modify through CSS. Sizes cannot be adjusted and cannot be moved.

What is the margin, border, and padding?

Margin is the space outside the box model. The border is the outline of the box that surrounds the element in the web page. Lastly, the padding is the space from the border going inwards the box model.

What are units of measurement you can use in CSS?

The different units of measurement that we can use in CSS are point for fonts, pixels, inches, centimeters, em, rem, and percentages.

When would you use a figure tag?

We use a figure tag when we want to make an image or any figure that is an inline tag act like a block tag so that we can modify it with block properties.
