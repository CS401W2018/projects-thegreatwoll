What is the difference between an ID and Class?

An ID is used to apply a style or modification to a specific and particular content within a tag. It can only be applied to one element in the HTML. On the other hand, a Class can be used in multiple tags or elements to modify the look or style of the webpage.

What is the difference between a SPAN tag and a DIV tag?

Both SPAN tags and DIV tags allow for customization through CSS but SPAN tags are in-line selectors. It is pretty much invisible to search engines. DIV tags are tags that separate it's contents from the others. It creates a division or segment from other parts of the HTML.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

DIV tags and the HEADER, MAIN, or FOOTER are close relatives with each other. Before header, main, or footer existed in HTML, div tags were widely used. The only difference is that a div tag is a general tag that can be used with any purpose while header, main, or footers have specific purposes as denoted by it's name.
