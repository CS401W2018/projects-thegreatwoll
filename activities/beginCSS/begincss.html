Inline CSS

Refers to the CSS that is put into the line of HTMLs for the web page you are creating. They are put into the tags of each part of the HTML and affects that particular tag. Takes number one priority over other CSS codes.

Embedded CSS

Refers to the CSS that is put into the head section of the HTML code. This affects specific tags or parts throughout the whole HTML depending on which tag you apply the customization to. Takes second priority after inline CSS.

External CSS

Refers to CSS that is written in an external file. Widely used form of CSS as it allows for global effect and uniformity throughout the website that you are designing.

Why External CSS?

We use external CSS as it allows uniformity in the design and look of the different pages that belongs to the same web site that you are creating. This creates general congruency and appeal to the website.

The Use of a Selector

A selector is used to select a particular section of the HTML and only apply the CSS code in that particular selection.
