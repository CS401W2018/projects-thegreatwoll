Internet

The internet refers to the network of computers connected together that exchange data and information.

Web

The term web refers to the contents in the internet that users can access.

How are the Internet and Web different?

The internet refers to the network of computers connected to share data and information while the web refers to the contents that are exchanged.

ASCII and Unicode

ASCII stands for American Standard Code for Information Interchange which is a standard that sets bits or numbers to represent characters of letters and symbols.

Unicode is an encoding standard that allows text or characters to be consistent accross devices and platforms.

What is UTF-8

UTF-8 stands for Unicode Transformation Format-8 and is used to represent characters with the use of one to four bytes.

URL

URL stands for Universal Resource Locator which refers to the unique address that is given to webpages on the internet so it can be specifically identified.

Domain Name

Domain Name refers to the understandable name that is given to a webpage.

HTTP

HTTP means Hypertext Transfer Protocol which refers to the protocol that controls how different hosts connect and exchange data with each other in a network or the internet.

HTTPS

HTTPS stands for Hypertext Transfer Protocol Secure and is the secured version of HTTP.

IP Address

IP Address is like the mailing address of a computer that is used to identify the host in a network. It changes depending on where or which network the host is connected to.

DNS

DNS stands for Domain Name Server/System that translates the domain names that are human readable to IP Addresses that are used by computers.

W3C Recommendations

W3C Recommendations is a set of standardized practices that are best used in web design or programming.

XML, XHTML, HTML, HTML5

These are file formats used in computers and the web.

XML - Extensible Markup Language is a human and machine readable fomat that allow people to store, transmit, and reconstruct data.

XHTML - Extensible HyperText Markup Language is a member of the XML family that incorporates the use of HTML.

HTML - Hypertext Markup Language is a widely used programming language for web development.

HTML5 - The newest version of HTML.
